# cube_robot_7x7x7

#### Description
使用树莓派控制的魔方机器人，可以在平均5-6分钟的时间内完成7阶魔方的还原。
使用C + Python开发，含有结构设计、软件部分，实物照片，演示视频。
算法使用rubiks-cube-NxNxN-solver，可还原任意阶数魔方，在原版基础上略有修改。
